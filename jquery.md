# Dom Manipulation 

Adding jQuery to the web is easy:
```html

<html>
    <head>
        <title>Demo page</title>
    </head>
    <body>
        <!-- your page -->
        <script src="[location of minified jQuery JavaScript file]"></script>
        <script>
            // your custom scripts that use jQuery
        </script>
    </body>
</head>
```
